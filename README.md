## VidHub 是什么？先把它说清楚

VidHub 是一款跨平台视频播放器，核心功能是把分散在网盘、NAS、本地、媒体服务器里的视频资源聚合到一处，自动抓取海报和剧集信息，做成一个私人版的"海报墙首页"。它支持 iOS、iPadOS、macOS、Apple TV 和 Android（手机/平板/电视），同一套媒体库能在多设备间同步，对阿里云盘、百度网盘这类国内网盘是原生直连，不需要额外折腾。

一句话总结：**VidHub = 海报墙整理 + 全平台播放器 + 国内网盘直连**，三件事打包在一个 App 里。

## VidHub 都能干什么？拆开看核心功能

能接入的资源来源列出来会更直观：

- 国内主流网盘：阿里云盘、百度网盘、115 网盘、移动云盘、123 云盘，直接在线播放，不用先下载到本地

- NAS / 网络存储：通过 SMB、WebDAV 协议连接群晖、极空间等设备

- 媒体服务器：直连 Emby、Jellyfin、Plex、飞牛 fnOS

- 本地文件：手机、电脑里存的视频，照常支持

播放器本身的底子也不差。MKV、MP4、AVI、RMVB、WMV、MOV、WebM 这些格式基本都能放，4K、HDR、杜比视界也支持，倍速、在线字幕搜索、截图、OCR 文字识别、视频调色、音频均衡器这些细节功能也齐全。

真正让人觉得"真香"的是自动刮削这一步。文件名稍微整理规范一点，VidHub 就会自动从 TMDB 一类数据库把海报、导演、演员、剧情简介、分集信息填好，剧集还会按季按集自动归类。打开主界面是一面排列整齐的海报墙——这也是它和普通播放器最大的差别。
